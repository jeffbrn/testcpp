# Matrix and Vector Library

## Google tests

### Setup

(CMake Quickstart)[https://google.github.io/googletest/quickstart-cmake.html]

```
$ cmake -S . -B build
$ cd build
$ cmake --build .
```

Test executable is now ready to run

### Run Tests


### Valgrind

```
$ valgrind -s --leak-check=full ./vector_tests
```
